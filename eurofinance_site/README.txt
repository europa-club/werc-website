# Euro Finance (WERC) — Static Website

This ZIP contains a complete, framework-free static website for **Euro Finance (WERC) / Wien Euro Reserve Coin**.

## Key compliance/clarity message (prominent on homepage)
- **Not Circle**
- **Not EURC**
- **Not euro-pegged stablecoin**
- **Market-priced digital asset**

## What’s inside
- `index.html` — homepage (includes the bold statement on top)
- `security.html` — security & audit page (PDF link included)
- `docs.html` — documentation page (whitepaper placeholder + brand kit)
- `privacy.html`, `terms.html`
- `assets/` — CSS/JS/images/PDF
- `robots.txt`, `sitemap.xml`

## How to deploy (any hosting)
1. Upload all files/folders to your hosting root (e.g., `/public_html`).
2. Open `sitemap.xml` and replace `https://YOUR-DOMAIN/` with your actual domain.
3. (Optional) If your host supports it, enable HTTPS.

## Notes
- No build step required.
- No external libraries required.
- Language toggle: EN/RU (top-right).
