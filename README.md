# Euro Finance (WERC) — Official Website

This repository contains the static website for Euro Finance (WERC) / Wien Euro Reserve Coin.

## IMPORTANT: Not Circle / Not EURC
Euro Finance (WERC) is an independent multi-chain DeFi project and is not affiliated with Circle.
WERC is not EURC, not a stablecoin, and not euro-pegged. The “€” symbol is thematic branding only.

Standalone disclosure page (for TrustWallet):
