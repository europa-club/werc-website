# Euro Finance (WERC)

## Official Statement

Euro Finance (WERC) is an independent multi-chain DeFi asset.

- NOT a stablecoin  
- NOT pegged to EUR  
- NOT affiliated with Circle or EURC  

The token is fully market-driven and operates without fiat backing or centralized control.
